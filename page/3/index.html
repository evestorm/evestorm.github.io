苹果app如何在其他人电脑上发版

发表于 2020-10-23 更新于 2020-12-17 分类于 移动端 ， iOS Changyan：

问题描述

之前只有我的电脑能发版，其他电脑上传不了；希望在其他电脑上也能发版。

解决方案

在能发版的电脑上 Xcode -> Preferences… -> Accounts -> 左下角齿轮 -> Export Account 。然后会导出一个文件，把这个文件传给另一台机器，然后安装就好了。

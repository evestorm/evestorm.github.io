Function函数笔记

函数声明

🌈 函数表达式（有名）

  • 函数表达式右侧函数有函数名，也会被系统忽略。外部无法调用

函数表达式（匿名）

形参、实参

🌈 形参、实参个数可以不相等

  • arguments.length 是 实参个数
  • [function name].length 是 形参个数

实参求和

🌈 arguments

  • 函数内部可以修改实参的值
  • 实参如果没有传入的形参，则这个形参在函数内部无法赋值

return

  • 终止函数的执行，return 后边的语句不会被执行
  • 返回相应的值

函数参数默认值

  • 函数参数默认值：undefined
  • 可以在定义形参时赋值默认值
  • 不想传的值，可用 undefined 占位代替
  • 兼容IE等旧版本浏览器

转载-进制转换

发表于 2021-10-07 分类于 前端 / JS / ECMAScript Changyan：

进制转换

二进制 → 十进制

将二进制的 101011 转换为十进制的步骤如下：

  1. 第 0 位：1 * 2 ^ 0 = 1;
  2. 第 1 位：1 * 2 ^ 1 = 2;
  3. 第 2 位：0 * 2 ^ 2 = 0;
  4. 第 3 位: 1 * 2 ^ 3 = 8;
  5. 第 4 位：0 * 2 ^ 4 = 0;
  6. 第 5 位：1 * 2 ^ 5 = 32;
  7. 读数：把结果值相加，1 + 2 + 0 + 8 + 0 + 32 = 43，即(101011)B=(43)D。

随着 github 上 star 的项目越来越多，找起来也越来越不方便。

Google 了一下发现个不错的网站能帮我们管理 star：[https://astralapp.com/](https://link.zhihu.com/?target=https%3A//astralapp.com/)

<!-- more -->

特性：

- GitHub 授权之后自动同步所有 Star 项目
- 可以使用标签分组（支持一个项目多个标签）
- 可以给每个 repo 添加自己的笔记
- 如果 Star 项目有变动可点击左上同步按钮同步

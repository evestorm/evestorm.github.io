如何在 github 上搭建博客 (for Mac)

发表于 2017-02-25 | 分类于 Hexo | | 阅读次数

拥有 GitHub Pages

  • 搭建 GitHub Pages 教程

安装 Hexo

  • 前提 需安装 Node.js , git （进入官网，根据提示下载安装即可）

  • 上面必备的两个程序安装完成后，来到终端

    • 执行：$ npm install -g hexo-cli

    • 如果报错如下：

      解决办法：执行 $ sudo chown -R $USER /usr/local

      输入你的电脑密码，然后再次执行 $ npm install -g hexo-cli

建站

  • 安装完成后 hexo init yourname //yourname是你文件夹名，随意起

  • 出现 INFO Start blogging with Hexo! 后

  • $ cd yourname

  • $ npm install

  • 进入你的文件夹目录 /Users/macbook/deskdrop/yourname/

  • 打开 _config.yml 文件，并滚动到最下面添加如下配置信息（删除原有的deploy和type字段，用下面4行代码替换）：

    deploy:
    type: git
    repo: https://github.com/yourusername/yourusername.github.io.git
    branch: master

    repo 地址为你 github page git地址 做完以上操作后，保存退出

  • 使用终端 cd 到你的 站点文件夹(/Users/macbook/deskdrop/yourname/) 使用hexo s命令，如果成功会打印类似Hexo is running at http://localhost:4000/. Press Ctrl+C to stop的一句话，再打开你的浏览器输入localhost:4000地址，到此为止你的本地站点已经建立完成。

发布

  • 执行如下命令：

    hexo clean // 清除缓存
    hexo g // 生成静态文件 g - generate
    hexo d // 部署网站 d - deploy

    (P.S. 详细指令参考：Hexo指令文档)

    • 如果出现：

      You should configure deployment settings in _config.yml first!

      Available deployer plugins:

      For more help, you can check the online docs: http://hexo.io/

      做这样一步操作： $ npm install hexo-deployer-git --save

      然后再次执行： $ hexo d 发布

  • 出现 INFO Deploy done: git 后代表部署成功
  • 打开你的 github page 查看吧~

element-ui全屏select下拉框不显示下拉列表的问题

发表于 2021-01-27 更新于 2022-11-12 分类于 前端 ， UI ， element-ui

问题

项目中需要用到页面全屏功能，使用后发现 element-ui 的 select 组件无法显示下拉列表。

原因

element-ui 默认将弹出框插入到了 body 元素。当设置了某个元素全屏时，会遮挡住原来的 select 下拉数据。

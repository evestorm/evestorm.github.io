发布/订阅者模式

对于 发布/订阅模式 你可以这样理解，一个人关注（订阅）了某个微信公众号，那么在此公众号发布文章的时候，这人就能收到文章的消息推送，而一般情况下一个公众号不止一个人关注，一旦发布文章，所有订阅了该公众号的人都会接收到通知。这就是发布订阅模式，而且它还是种一对多的映射关系。其实这种模式大家应该也一直在用，比如说事件监听，当你需要在点击某个按钮后触发一些行为时，就用到了 发布/订阅模式 ：

结语

了解上面两个概念非常重要，因为 Vue 就是在内部实现了一个 发布/订阅者模式 ，当你每次去 set 的时候，就会触发一个 trigger 来提交事件，告诉大家当前属性更新了，通过 get 来注入订阅者，当属性发生变化后这些订阅者就能收到“通知”，这便是 Vue 实现数据响应式的一个大概原理。
